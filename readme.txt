Search on Wordpress site using Telegram Inline Bots.

== Description ==

With the new in-line mode, bots become omnipresent and can be used as a tool in any of your chats, groups or channels â€“ it doesn't matter, whether the bot is a member or not.

Telegram Inliner (Telegram Search Plugin) helps you to integrate Telegram inline bot to your Wordpress site and receive search results.

first of all you have to <a href="https://core.telegram.org/bots#botfather">Create a New Bot</a> and get your Token API, then setinilne mode for your bot.

Features:

*   search in custom post types
*	understand which results gets more clicks by users
*	No results Template
*	Message Template


Note: this plugin is working only using webhook and needs a free account from https://teletter.net

== Installation ==
From your WordPress dashboard
1. Visit 'Plugins > Add New'

2. Search for 'Telegram Inliner'

3. Activate Telegram Inliner from your Plugins page. (You'll be greeted with a Welcome page.)

From WordPress.org
1. Download Inliner.

2. Upload the 'Telegram Inliner' directory to your '/wp-content/plugins/' directory, using your favourite method (ftp, sftp, scp, etc...)
Activate Telegram Inliner from your Plugins page.

Extra
Visit 'Inlier Settings' and adjust your configuration.

a new Page Template is added as 'Inliner Page', make a new page, use this template set your Webhook to this page.
